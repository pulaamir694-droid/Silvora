# Silvora Accessories 🌟

متجر إكسسوارات فاخر مبني بـ React + Supabase

## 🚀 تشغيل المشروع

```bash
npm install
npm run dev
```

## ⚙️ الإعداد

1. أنشئ مشروع على [supabase.com](https://supabase.com)
2. شغّل الـ SQL الموجود في `SUPABASE_SETUP.md`
3. أنشئ ملف `.env` من `.env.example` وضع فيه بياناتك
4. أضف يوزر أدمن من Supabase → Authentication → Users

## 📁 البنية

- `src/pages/` — صفحات الموقع
- `src/services/` — التواصل مع Supabase
- `src/context/` — Auth و Cart و Toast
- `src/components/` — مكونات مشتركة
